###################
API Key の作成手順
###################

STEP1 Rescaleへのログイン
============================

Rescale へログインします。

- https://platform.rescale.jp/


STEP2 API Key の作成
============================

Rescale ログイン後下記を実施します。

1. アカウントをクリックします
2. APIをクリックします
3. APIキーの作成をクリックします
